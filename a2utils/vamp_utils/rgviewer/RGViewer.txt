
Resource Game Viewer v1.0

Программа предназначена для извлечения и\или дальнейшего просмотра ресурсов из игр.



Данная версия:

- использует плагины, причём плагины имеют расширение .dll и начинаются с RGVF (это плагины для паков и архивов) и RGVV (это плагины для просмотра)

- имеет файл конфигурации, в котором сохраняются размеры окна, вид панелей, пути для плагинов и извлечения



Пока не реализовано:

- строка статуса полностью безжизнена

- извлечение производится по существующим какталогам только, по папкам форматов не реализовано

- нет сортировки



Работа с программой:

- сначала необходимо настроить опции(панель -опции-). Здесь по-моему всё интуитивно понятно.

- далее необходимо выбрать формат открываемого файла- это панель -выбор-, левое окно. Нужно просто указать мышью и в правом окне появится инфо об этом формате

- после можно здесь же испорльзовать кнопку открыть или воспользоваться меню

- после выбора корректного файла при отсутсвии ошибок программа открывает панель -список- с файлами, содержащимися в паке\архиве



Работа со списком:

- в панели -извлечение- можно извлечь все файлы или выделеные

- выделяются файлы мышью в панели -список- или поиском. При поиске строки учитывается регистр.

- по выделеным файлам можно осуществить переход(визуальный с перемещением фокуса списка) с помощью кнопки -переход-



Просмотр файлов:

- на вкладке просмотр при установленном флажке -автоматический просмотр- ведётся поиск нужной библиотеки по расширению файла

- при снятом флажке необходимо указать в левом окне панели -просмотр- тип файла, после чего в правом окне появится список расширений, поддерживаемый плагином. Далее нужно 2 раза нажать в панели -список- на интересующем файле, принудительно указывая выбранный формат( как например: для прослушивания музыки в Heroes IV - там все файлы имеют расширение h4d - можно использовать такой способ)
