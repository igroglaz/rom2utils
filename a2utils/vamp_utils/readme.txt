a2hat.zip (221 kb, Автор: Nival) - Хэт для Аллодов 2 (русский вариант). Предназначен для совместной работы с a2server. Поставляется вместе с переизданием Аллодов 2.

a2server.zip (1,2 Mb, Автор: Nival) - Сервер для Аллодов 2 (русский вариант). Для дизайнеров аллодовских карт представляет интерес тем, что позволяет их тестировать. Особенно полезен для диагностики карт, которые вызывают зависания и сообщения об ошибках в игре или редакторе. Рекомендуется проверять все свои творения этой программой. Для корректной работы файлы из архива надо поместить в папку с игрой, а тестируемую карту прописать в server1.cfg. Запускать программу следует с помощью bat-файла. Ошибки выдаются на экран и в текстовый log-файл. Поставляется вместе с переизданием Аллодов 2.

grv.zip (400 kb, Автор: Александр Арефьев) - Game Resource Viewer v. 0.23beta, универсальный распаковщик ресурсов. В комплект входят модули для Аллодов, Аллодов 2 и других игр.

mfc.zip (53 kb, Автор: Михаил Бесчетнов) - Multi-format Converter v. 1.01, аналогичный предыдущему, но более универсальный распаковщик ресурсов. В комплект входит plug-in для Аллодов и Аллодов 2. Поддерживает форматы: *.res, *.16а и *.256.

rgviewer.zip (420 kb, Автор: Executor) - Resource Game Viewer v1.0, универсальный распаковщик ресурсов. Поддерживает Аллоды, Аллоды 2, Проклятые Земли и что-то там еще.

asources.zip (143 kb) - Исходники некоторых представленных в этом разделе утилит.
