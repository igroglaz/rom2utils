Если у вас проблемы с курсором и игра лагает, то распакуйте в папку с игрой DDwrapper выставив для allods2.exe совметимость с Windows XP SP3.

Устраняет проблемы с курсором и лагами клиента на Windows 7 и выше, поднимает FPS. Данную утилиту нужно распаковать в папку с игрой и запустить allods2.exe в режиме совместимости с Windows XP SP3.

Внимание!!! Утилита несовместима с Редактором карт 3.0.
