Shows base skills and characteristics + each skill progress.
Unpack it to your RoM2 directory. It's better to run it either on second monitor or with RoM2 windowed mode

To faster updating of character info (each second instead one time in 15 seconds), you should patch RoM II with that.
Usage:
charinfo.exe [--en | --ru]

--en: force using english interface
--ru: force using russian interface

Author: Vladimir Chebotarev aka ex-lend
Version: 0.20
