a2helper
Author: Vladimir Chebotarev aka ex-lend
Categories: RoM 2, Common
Version: 0.2

RoM II helper utility.

Provides features:
inventory scrolling with PgUp/PgDn keys (for allods2.exe only);
commands history (for allods2.exe only).
Before use you should modify your client with following patch:

(in subfolder)

allods2.exe
000001C7: 40 C0

To use it simply run it before or after starting RoM II.
