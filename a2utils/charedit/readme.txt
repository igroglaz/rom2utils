This tool allows to edit .a2c and .lgn files

Usage:
charedit.exe [--en | --ru] [--anet] <some_character.a2c | some_login.lgn>
--en: force using english interface 
--ru: force using russian interface 
--anet: treat character id 2 as access rights (Allods.NET mode)

Example:

charedit.exe 1234567890.a2c
