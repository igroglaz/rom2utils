
 Данный архив содержит две экранные темы (screen theme) по игре
"Аллоды 2: Повелитель душ".
 Установка данных тем требует наличия пакета Microsoft Plus! на
вашем компьютере.

 Для установки распакуйте его в директорию со стандартными темами
 экрана (обычно это "c:\Program Files\Plus!\Themes"), откройте 
control panel, два раза нажмите левой клавишей мыши на иконке 
"Desktop Temes", в открывшемся окне выберете один из двух выриантов
("Allods2 var 1" или "Allods2 var 2"), после чего нажмите на клавишу
"Ok".

ВНИМАНИЕ!
 - НЕ ИСПОЛЬЗУЙТЕ DOS-версию pkunzip для распаковки данного архива!
 - Не переименовывайте директорию с темами.
 - В архиве уже записан стандартный путь к нужным директориям,
   если Вы используете WinZip, убедитесь что выбран режим
   "Use folder names" при распаковке.
