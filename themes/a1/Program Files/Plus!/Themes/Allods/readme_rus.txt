
 Данный архив содержит две экранные темы (screen theme) по игре
"Аллоды: Печать Тайны".
 Установка данных тем требует наличия пакета Microsoft Plus! на
вашем компьютере.

 Для установки распакуйте его в поддиректорию \Allods
директории со стандартными темами экрана (обычно это 
"c:\Program Files\Plus!\Themes"), откройте control panel, два раза
нажмите левой клавишей мыши на иконке "Desktop Temes", в 
открывшемся окне выберете Theme: other, затем один из двух файлов
(Allods.theme или Allods2.theme), после чего нажмите на клавишу
"Ok".

ВНИМАНИЕ!
 - НЕ ИСПОЛЬЗУЙТЕ DOS-версию pkunzip для распаковки данного архива!
 - Не переименовывайте директорию с темами.
 - В архиве уже записан стандартный путь к нужным директориям,
   если Вы используете WinZip, то можно просто распаковать его в
   корневую директорию диска, убедившись что выбран режим
   "Use folder names" при распаковке.
